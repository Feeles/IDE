3 | 一番下の列の数（横） | 0 ~ 
4 | 中央の段の数（タテ） | 0 ~ 
5 | 横方向の間かく | 0 ~ 
6 | タテ方向の間かく | 0 ~ 
7 | オブジェクトの横幅 | 0 ~ 
8 | オブジェクトの高さ | 0 ~
