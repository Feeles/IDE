# 布

面に並べたスタックをソフトボディにすると、布のような表現ができるよ  
ただし、全てを自由にすると床に落ちてしまうので、  
上の行 (0 ~ 20 番まで) のオブジェクトだけは固定したんだ

isStatic | 静止するかどうか | フラグ | false | true
